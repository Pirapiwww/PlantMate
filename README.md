# PlantMate

A. Deskripsi Aplikasi

PlantMate adalah aplikasi pendamping perawatan tanaman untuk membantu pengguna untuk
persiapan penanaman, menanam, serta merawat tanaman dengan lebih mudah, dan akurat. 
Aplikasi ini memberikan hasil rekomendasi AI dari form yang pengguna input untuk mengetahui
apakah proses ini sudah sesuai dan AI juga akan memberikan insight tambahan untuk mendukung
proses tersebut.

B. Fitur Aplikasi
	
	1. Plant Journal: 
		Journal pendamping persiapan penanaman, penanaman, serta perawatan yang terintegrasi AI.
	2. Plant Lens: 
		Analisis photo tanaman untuk mengetahui persen sehat tanaman yang terintegrasi ML.
	3. Plant Encyclopedia: 
		Kumpulan ensiklopedia tanaman dengan informasi seputar penyiraman, paparan sinar matahari, dan pemangkasan.
	4. Plant News: 
		Kumpulan berita terbaru seputar tanaman dan sejenisnya dalam seminggu terakhir.

C. Integrasi

	- Groq (Analisis AI)
	- Google ML Kit Labeling (Labeling untuk gambar)
	- Google ML Kit Translate (Translate 2 bahasa eng-ind)
	- PlantId API (API berbasis ML untuk analisis gambar)
	- Perenual API (API untuk ensiklopedia tanaman)
	- Google News RSS (RSS untuk berita terkini seputar tanaman)

D. Cara Penggunaan

	1. Plant Journal:
		Tambahkan / isi journal dan dapatkan rekomendasi AI.
	2. Plant Lens:
		Ambil foto tanaman dan sistem akan menilai kesehatan tanaman.
	3. Plant Encyclopedia:
		Cari tanaman yang ingin diketahui informasinya.
	4. Plant News:
		Lihat berita terbaru terkait tanaman.

D. Instalasi

	1. Clone repository ini / download sebagai ZIP
	2. Bukan project menggunakan Android Studio atau sejenisnya
	3. Masukkan Groq API Key ke dalam object GroqApiKey yang pathnya sejajar dengan MainActivity.kt
